Firebase Setup

Apna Firebase project ka config paste karo. Data Firestore mein securely store hoga — kisi bhi device se access kar sakte ho.

Firebase setup kaise kare (ek baar):
1. console.firebase.google.com pe jao, new project banao.
2. Project settings — "Your apps" — Web app add karo (</>).
3. Firebase SDK config wala section mein apiKey, authDomain, projectId, appId milega — wahi yahan paste karo.
4. Firestore Database — Create database — Start in test mode (ya production rules set karo).
5. Bas ho gaya, data automatically sync hoga!
