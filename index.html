Firebase Setup

Paste your Firebase project config below. Data will be securely stored in Firestore and accessible from any device.

How to set up Firebase (one time only):
1. Go to console.firebase.google.com and create a new project.
2. Project settings — "Your apps" — Add a Web app (</>).
3. Copy the apiKey, authDomain, projectId, appId from the Firebase SDK config section and paste them here.
4. Firestore Database — Create database — Start in test mode (or set production rules).
5. Done! Data will sync automatically across all devices.
